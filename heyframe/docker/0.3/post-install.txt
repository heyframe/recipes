  * <fg=blue>Setup</>
    This recipe has created a <comment>docker/Dockerfile</comment> with an example file to build a HeyFrame Docker image.
    You can build it with <comment>docker build -t <name> -f docker/Dockerfile .</comment>.
    Make sure you to set a correct `APP_SECRET`, you can generate one with `openssl rand -hex 128`
    See https://github.com/heyframe/docker for the complete documentation to configure things.
